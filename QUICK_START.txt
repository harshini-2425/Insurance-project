## ❌ Troubleshooting

### Admin Cannot Login
- Check email spelling: `elchuritejaharshini@gmail.com`
- Check password: `958181630`
- Verify backend is running on port 8000

### Cannot Access /admin Routes
- Verify using admin credentials (not regular user)
- Check browser console for errors
- Clear cookies/cache if needed

### "Access Denied" Error
- User is not admin - login with admin credentials
- JWT token expired - login again
- Check database: user should have `role='admin'`

### Backend Not Responding
- Run: `python -m uvicorn backend.main:app --reload`
- Check port 8000 not in use
- Verify PostgreSQL is running

### Frontend Not Loading
- Run: `npm run dev` in frontend-react folder
- Check port 5175 not in use
- Clear browser cache
